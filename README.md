## CryptoMonitor API

### Exposed endpoints
```
/coins 
/coins/price/:coinId
```


### Copy sample.env to .env and change values

## Installation

### With Docker and Docker compose
```
docker compose up --build
```


### Locally (without docker)
```
npm i 
npm start
```
